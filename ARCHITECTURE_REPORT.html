Ecosistema BTraffic

Análisis técnico sobre la arquitectura de captura, cualificación y orquestación de activos digitales financieros.

El Porqué y Para Qué

BTraffic no es una web corporativa; es una Fábrica de Activos Digitales. Cada componente está diseñado para transformar el tráfico frío en capital medible, eliminando la fricción humana mediante orquestación de IA.

Propósito Principal

Demostrar autoridad técnica instantánea mediante diagnósticos forenses (Sherlock) y automatizar el embudo de ventas sin intervención manual.

Valor Estratégico

Convertir el sitio web de un "gasto de marketing" en un "activo financiero" que opera con lógica 80/20 y escala infinitamente.

Arquitectura Técnica

Core Framework

Next.js 15.2 (Latest Stable) con React 19. Optimizado para Core Web Vitals y carga bajo los 800ms.

App Router Server Actions TypeScript Strict

Motores de Inteligencia

Enjambre de modelos distribuidos para análisis de velocidad e inteligencia de cualificación.

Groq (Llama 3.3 70B) Gemini 1.5 Flash

Persistencia y Tracking

Base de datos de activos para leads y sistema de logs forenses para seguimiento de ROI.

Supabase n8n (Webhooks) Vercel Edge

Funcionalidad & Orquestación

01

Diagnóstico Forense (Sherlock Engine)

Motor de auditoría en tiempo real que escanea SSL, tiempos de carga, píxeles de Facebook y tags de Google. Genera un reporte dinámico de "Fugas de Capital" que impacta emocionalmente al lead.

02

Cualificación Inteligente (Brain)

Usa IA para clasificar al lead según su cargo, ingresos declarados y "pain point". Calcula un Score de 0-100 para priorizar la atención comercial.

03

Orquestación de Activos

Sincronización instantánea con n8n y Supabase. El lead entra al CRM y recibe un email forense personalizado sin que nadie haya movido un dedo.
